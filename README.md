## 📌 Project Name
**Friends Vibe**

🌟Friends Vibe is a real-time communication platform that allows users to chat, make voice calls, and video calls, creating a smooth and friendly space for staying connected with friends.


## 🛠️ Technologies to Use
- Next.js (React Framework)
- Tailwind CSS
- Context API / State Management
- HTML-5
- CSS_3
- CSS-JS (ES-6)
- React Icons
- Toistify!
- daisyUI



## 🚀 Features

- 💬 1. Real-time Chat
- 📞 2. Voice Calling
- 🎥 3. Video Calling
- Sorting System :
(1)sort by title
 (2)sort by time
- Real time Call Logs
- Smart Dassboard (paichart ) system
